# water_ripple_simullation
  an interactive water ripple effect using OpenGL! 


command to run:
      do install:sudo apt-get install freeglut3-dev
run with
      g++ water_ripple.cpp -o water_ripple -lGL -lGLU -lglut
      ./water_ripple
